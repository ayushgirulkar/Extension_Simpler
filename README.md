# 🚀 Extension_Simpler

A smart Chrome extension that enhances your browsing experience using **AI-powered tools** like translation, synonyms, and text summarization.

---

## ✨ Features

### 🌐 Text Translation

* Translate selected text into:

  * Hindi 🇮🇳
  * Marathi 🇮🇳
* Fast and accurate results using AI

### 📖 Synonyms Finder

* Get synonyms for selected words instantly
* Helps improve vocabulary and writing quality

### ✂️ Summary Generator

* Convert long paragraphs into short summaries
* Ideal for students and quick understanding

---

## 🧑‍💻 How It Works

1. Select text on any webpage
2. Open the extension popup
3. Choose:

   * Translate
   * Synonyms
   * Summary

## 📦 Installation

```bash
git clone https://github.com/ayushgirulkar/Extension_Simpler.git
```

### Load Extension in Chrome

1. Open Chrome → `chrome://extensions/`
2. Enable **Developer Mode**
3. Click **Load Unpacked**
4. Select the project folder
